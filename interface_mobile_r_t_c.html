Detailed Description

MobileRTC class is a class that exposes an API Rest Client. Access to this class and all other components of the MobileRTC can be granted by including <MobileRTC/MobileRTC.h> in your source code.

Warning
This class provides a class method sharedSDK which provides a preconfigured SDK client.
